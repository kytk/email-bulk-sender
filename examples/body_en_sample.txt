Email Bulk Sender Test - {所属} {氏名}

Dear {氏名} at {所属},

This is a sample template for the "Email Bulk Sender" tool.

## How to Write a Template
This file follows these rules:

  Line 1: Subject (this line becomes the email subject)
  Line 2: Empty line (required)
  Line 3 onwards: Body

## About Placeholders
The following strings in the template are automatically replaced
with each recipient's information:

  {所属} -> Value from the "所属" (affiliation) column
  {氏名} -> Value from the "氏名" (name) column

For example, if the recipient list contains:
  所属 (affiliation): Sample University
  氏名 (name): Taro Yamada

The subject of this email becomes:
"Email Bulk Sender Test - Sample University Taro Yamada"

## Recipient List Format
Prepare a CSV (.csv) or Excel (.xlsx) file with the following headers:

  所属,氏名,メールアドレス
  (or: affiliation,name,email)
